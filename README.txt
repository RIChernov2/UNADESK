Необходимо написать библиотеку, в которой будет один метод,
который по трем сторонам треугольника будет определять,
является треугольник остроугольным, прямоугольным или тупоугольным.
Метод должен работать максимально быстро. Длины могут изменяться нецелых
числах. Результат положить на гитхаб.  Тесты тоже приветствуются.

Info:
Остроугольный треугольник — это треугольник, в котором каждый угол острый.
Прямоуго́льный треуго́льник — это треугольник, в котором один угол прямой (то есть 90 градусов).
Тупоугольный треугольник – это треугольник, у которого один угол тупой (то есть имеет градусную меру больше 90º).
